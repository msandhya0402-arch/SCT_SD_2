# Number Guessing Game

This is a simple Python project that generates a random number between 1 and 100. The user tries to guess the number, and the program tells whether the guess is too high, too low, or correct.

## Features

* Generates a random number
* Accepts user input
* Gives feedback on the guess
* Easy to understand and use
